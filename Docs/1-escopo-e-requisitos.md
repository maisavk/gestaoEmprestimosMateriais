# 1. Escopo e Requisitos do Sistema
## 1.1. Objetivo Geral
Sistema em C# para gerenciar chamados de suporte técnico usando Herança e Polimorfismo.
## 1.2. Requisitos Funcionais (RF)
* **RF01:** Cadastro de chamados especializados (Infraestrutura, Sistema Interno e Segurança).
* **RF02:** Cálculo dinâmico de prazo com base na prioridade e regras de desconto.
* **RF03:** Fechamento seguro de chamados mudando o status para Encerrado.
